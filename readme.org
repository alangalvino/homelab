#+TITLE: Homelab Documentation

* Overview
[[https://github.com/alangalvino/homelab/raw/main/.assets/images/homelab.png]]

This homelab consists of:
  - Three Mac Mini nodes running Proxmox VE
  - Synology NAS for shared storage
  - Home Assistant running as a dedicated VM
  - One Debian VM at =192.168.50.100= acting as the container repository host
  - Podman compose stacks deployed with Terraform (Pi-hole, Deye Dummycloud, Homepage)
  - Environment automation with Ansible and Terraform

* Installation
To set up this homelab from scratch, follow the step-by-step guide in [[https://github.com/alangalvino/homelab/blob/main/install.org][install.org]].

* Networking
** Overview
- All infrastructure is on subnet =192.168.50.0/24=, except Home Assistant on IoT VLAN (=192.168.40.0/24=).
- Container services are exposed from the Debian VM at =192.168.50.100=.

** Static IP Addressing Scheme
| Device/Service                    |     IP Address |
|-----------------------------------+----------------|
| Synology NAS                      |  192.168.50.10 |
| Mac Mini #1 (Proxmox)             |  192.168.50.11 |
| Mac Mini #2 (Proxmox)             |  192.168.50.12 |
| Mac Mini #3 (Proxmox)             |  192.168.50.13 |
| Container repository VM (Debian)  | 192.168.50.100 |
| Home Assistant                    |  192.168.40.40 |

** Services on 192.168.50.100
| Service         | Protocol/Port | URL/Usage                        |
|-----------------+---------------+----------------------------------|
| Homepage        | TCP/80        | http://192.168.50.100            |
| Pi-hole Web     | TCP/8080      | http://192.168.50.100:8080/admin |
| Pi-hole DNS     | TCP/53 UDP/53 | LAN DNS server                   |
| Deye Dummycloud | TCP/10000     | Deye inverter cloud endpoint     |
